# Project-5-Traffic-Light-Modded
Replica of traffic light using a set of east and west lights

Begins by identifying pins and buttons with their respective pin along with traffic flow times and delays to reduce redundancy.
Boolean variables for traffic west and east are also setup to allow for checking of button inputs

In the setup serial begin is setup to print and INPUTS and OUTPUTS for LEDS and buttons, with west side traffic set up first to go.

In the loop IF statements are made for three buttons East, West, and pedestrian to manipulate to order of traffic. Turns respective side on along with a for loop to allow for the blinking of lights.
